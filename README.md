# 🚀 Evently

Evently es una plataforma web desarrollada con **FastAPI** para la gestión de eventos universitarios.

Permite:
- Crear eventos
- Gestionar inscripciones
- Controlar aforo automáticamente
- Visualizar cupos disponibles en tiempo real

---

# 🛠️ Tecnologías

- Python 3.13
- FastAPI
- SQLAlchemy
- SQLite
- Jinja2
- HTML/CSS/JavaScript
- Render

---

# ⚙️ Instalación

```bash
git clone https://github.com/Sarayssd/Evently.git
cd Evently
pip install -r requirements.txt
uvicorn main:app --reload
```
